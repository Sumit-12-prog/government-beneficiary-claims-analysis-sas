# Government Beneficiary Claims Analysis using SAS

A healthcare claims analytics project developed using **SAS** to analyze beneficiary claim data, generate key performance indicators (KPIs), and design a professional management dashboard inspired by SAS Visual Analytics.

---

## Project Overview

This project focuses on analyzing government healthcare beneficiary claims to understand claim trends, hospital performance, package utilization, approval statistics, and beneficiary distribution.

The project includes:

- Data Cleaning
- Data Transformation
- KPI Generation
- SQL Analysis
- Statistical Analysis
- Dashboard Design

---

## Technologies Used

- SAS
- PROC SQL
- PROC MEANS
- PROC FREQ
- DATA Step Programming
- SAS Visual Analytics (Dashboard Design)

---

## Dataset

The dataset contains healthcare beneficiary claim records with information such as:

- Beneficiary Details
- Hospital Information
- Claim Amount
- Approved Amount
- Package Specialty
- Policy Year
- TPA
- Hospital Type
- Beneficiary District
- Submission & Approval Dates

---

## Data Preparation

The following preprocessing steps were performed:

- Imported Excel dataset into SAS
- Handled missing values
- Standardized beneficiary names
- Cleaned package information
- Calculated approval percentage
- Calculated processing delay
- Created derived analytical variables
- Generated monthly claim data

---
